# GitHub Pages Deployment

To deploy this website using GitHub Pages:
Go to Settings -> Pages -> Build and deployment -> Source: Deploy from a branch -> Branch: master -> Folder: /docs -> Save.

## Files

- `index.html`: static homepage for the project blog/demo.
- `assets/style.css`: responsive styling.
- `assets/figures/`: copied and regenerated figures used by the page.

No backend, package manager, build step, or server is required.
